# 📈 Crypto Dashboard (Streamlit + MongoDB)

Dashboard de criptomoedas com atualização em tempo real, histórico de preços
e conversão automática de USD para BRL.

Projeto desenvolvido em Python utilizando Streamlit, MongoDB e APIs públicas.

---

## 🚀 Funcionalidades

- 📊 Preços atuais de Bitcoin e Ethereum
- 🔄 Atualização manual e automática
- 💱 Conversão automática de USD para BRL
- 📈 Gráficos de evolução de preços
- 🗂 Histórico persistido no MongoDB
- 🧼 Tratamento de dados inválidos

---

## 🛠 Tecnologias Utilizadas

- Python
- Streamlit
- MongoDB
- Pandas
- Requests
- CoinGecko API
- Frankfurter API
